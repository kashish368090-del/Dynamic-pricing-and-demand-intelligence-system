# Dynamic Pricing and Demand Intelligence System

## 📌 Project Description
This project adjusts product prices dynamically based on demand using MySQL.

## 🛠️ Technologies Used
- MySQL
- MySQL Workbench

## 📂 Tables Used
1. Products (product_id, base_price, current_price)
2. Sales (sale_id, product_id, quantity, sale_date)

## ⚙️ Working
- Demand calculated using SUM and GROUP BY
- Prices updated using CASE statement

## 📊 Output
- High demand → price increase
- Low demand → price decrease

## 🚀 Features
- Dynamic pricing
- Demand analysis
- Real-time SQL updates
